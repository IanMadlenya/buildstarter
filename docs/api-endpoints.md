# API Endpoints

## HTML API

### Root

- `GET /` - loads Splash page

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Projects

- `GET /api/projects`
- `POST /api/projects`
- `GET /api/projects/:id`
- `PATCH /api/projects/:id`
- `DELETE /api/projects/:id`

### Investments

- `GET /api/investments`
- `POST /api/investments`
- `GET /api/investments/:id`
- `PATCH /api/investments/:id`
- `DELETE /api/investments/:id`
